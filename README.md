# Student Management System API

A simple ASP.NET Core Web API built for practical demonstration.

This project demonstrates how to build a clean, layered architecture using:

- ASP.NET Core Web API
- Entity Framework Core
- MySQL
- Repository Pattern
- Service Layer
- Request & Response Models (DTOs)
- Dependency Injection
